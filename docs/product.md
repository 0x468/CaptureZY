# 产品范围

## 目标

CaptureZY 的目标是成为一个以 Windows 为核心、原生感强、延迟低、桌面行为可预期的截图与贴图工具。

## 目标用户

- 高频截图的开发者、设计师与内容工作者
- 需要把参考图片长期固定在屏幕上的重度桌面用户
- 在意全局热键、托盘流程与低资源占用的 Windows 用户

## V0.1 范围

- 区域截图
- 全屏截图
- 将截图复制到剪贴板
- 将截图保存到文件
- 将截图固定到屏幕
- 基础贴图交互：移动、缩放、关闭、复制、保存
- 系统托盘与全局热键
- 基础多显示器与 DPI 支持

## V0.1 明确不做

- 云同步
- OCR
- 屏幕录制
- 协作分享
- 插件系统
- 复杂标注工具集

## 产品原则

- Windows 行为优先于跨平台诉求。
- 截图与贴图流程应保持快速、直接。
- 空闲资源占用应尽量克制。
- 特殊窗口与桌面集成应呈现原生行为，而不是明显的框架感。
