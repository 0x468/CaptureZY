# 路线图

## 阶段 0：基础设施

- 建立 CMake 工程布局
- 明确产品范围与架构边界
- 创建最小 Win32 应用壳层
- 补齐格式化、静态检查与仓库卫生文件

## 阶段 1：截图原型

- 实现原生应用启动与托盘初始化
- 打通截图覆盖层原型
- 接入首个截图主路径
- 验证基础剪贴板与保存流程
- 当前已具备区域截图、全屏截图、剪贴板复制与 PNG 保存基础能力

## 阶段 2：贴图原型

- 创建贴图窗口类型
- 实现移动、缩放、关闭与复制操作
- 实现右键菜单中的 PNG 保存入口
- 验证层级、置顶行为与多显示器移动

## 阶段 3：Alpha 可用性

- 已完成：设置界面
- 已完成：可配置热键
- 已完成：启动与基础持久化
- 已完成：崩溃日志与诊断输出

## 阶段 4：Beta 打磨

- 已完成：主窗口已收敛为隐藏宿主窗口
- 已完成：截图时支持右键取消
- 已完成：托盘单击/双击默认动作配置
- 截图交互设置与收敛
- DPI 边界问题
- 显示器热插拔行为
- 性能剖析
- 已完成：手工回归清单
- 已完成：基础 ZIP 发布打包入口
- 安装器与发布包装

## 阶段 5：智能选区增强

- 增加窗口级自动预框选
- 修正窗口预选与实际可见边界的一致性
- 增加任务栏与系统壳窗口识别
- 支持一键选择全屏
- 评估控件级预框选能力与兼容性边界
- 补充智能选区的回归清单

### 建议实现顺序

1. 窗口级自动预框选
2. 修正窗口可见边界与任务栏识别
3. 全屏快捷选择，例如 `Ctrl+A`
4. 截图交互设置收敛
5. 控件级预框选与高亮
