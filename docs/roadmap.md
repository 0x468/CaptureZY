# 路线图

## 阶段 0：基础设施

- 建立 CMake 工程布局
- 明确产品范围与架构边界
- 创建最小 Win32 应用壳层
- 补齐格式化、静态检查与仓库卫生文件

## 阶段 1：截图原型

- 实现原生应用启动与托盘初始化
- 打通截图覆盖层原型
- 接入首个截图主路径
- 验证基础剪贴板与保存流程
- 当前已具备区域截图、全屏截图、剪贴板复制与 PNG 保存基础能力

## 阶段 2：贴图原型

- 创建贴图窗口类型
- 实现移动、缩放、关闭与复制操作
- 实现右键菜单中的 PNG 保存入口
- 验证层级、置顶行为与多显示器移动

## 阶段 3：Alpha 可用性

- 已完成：设置界面
- 已完成：可配置热键
- 已完成：启动与基础持久化
- 已完成：崩溃日志与诊断输出

- 已完成：主窗口已收敛为隐藏宿主窗口

- 已完成：截图拖拽选区刷新与闪烁优化
